# Thailand Hospitals Directory for Myanmar Patients

A simple GitHub-ready project containing a static HTML directory of well-known hospitals in Thailand for Myanmar patients.

## Included
- Hospital names
- Phone numbers
- Official websites
- Address / area
- Myanmar interpreter/coordinator notes
- Specialist search guidance
- Regional hospitals
- Appointment booking tips
- Communication guidance for email / LINE / WhatsApp

## Files
- `index.html` — Main hospital directory page
- `README.md` — Project documentation

## Use Cases
This project is useful for:
- Myanmar patients planning treatment in Thailand
- Family members helping patients find hospitals
- Medical travel assistants
- Quick hospital reference on GitHub Pages

## How to Use
1. Clone or download this repository
2. Open `index.html` in your browser
3. Or publish it using GitHub Pages

## GitHub Pages Deployment
1. Push the project to a GitHub repository
2. Go to **Settings**
3. Open **Pages**
4. Select branch: `main`
5. Select folder: `/root`
6. Save
7. GitHub will provide a public URL

## Important Note
Hospital contact information and support channels may change over time.
Always verify:
- phone number
- address
- specialist availability
- consultation fees
- interpreter support
on the official hospital website.
